# marketing_fact_dashboards
📑 Power BI Project README
Project Title: Marketing & Customer Insights Analysis

📂 Data Source
Excel Dataset: Contains customer demographics, product sales, marketing campaigns, and revenue details.

Columns include: Customer_ID, Age, Gender, Region, Product, Campaign, Revenue, Conversions, Marketing Cost.

🎯 Project Objective
Analyze customer revenue by demographics (age, gender, product).

Evaluate marketing campaign performance (cost vs revenue, conversions, CTR).

Compare device-wise and channel-wise performance.

Build interactive dashboards for business decision-making.

🛠️ Tools & Technologies
Excel → Raw dataset preparation

Python (pandas, NumPy) → Basic data cleaning & analysis

Power BI → Interactive dashboards & visualizations

📈 Key Insights
Male customers contributed slightly higher revenue (52%).

Perfume generated the highest revenue among products.

Google Ads was the most effective marketing channel.

Mobile devices had higher conversions compared to desktop.

“New Arrival” campaign performed best in terms of revenue and conversions.

CTR exceeded target by 275%, showing strong ad engagement.
